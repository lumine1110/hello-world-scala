# 🌟 HelloWorld Project

![Scala](https://img.shields.io/badge/scala-3.3.5-red)
![License](https://img.shields.io/badge/license-MIT-blue)

Scala 3 の世界へようこそ！🚀
このプロジェクトは、Scala 3.3.5 を使用したシンプルかつモダンなスタータープロジェクトです。

---

## 📋 概要

`HelloWorld` は、Scala 3 の新しい構文と機能を体験するための最小限の構成を提供します。
`@main` メソッドを使用したエントリーポイントなど、Scala 3 のシンプルさを体感してください。

## 🛠️ 技術スタック

*   **言語:** [Scala 3.3.5](https://www.scala-lang.org/) 💎
*   **ビルドツール:** [sbt](https://www.scala-sbt.org/) 🔨

## 🚀 実行方法

以下のコマンドでプロジェクトを実行できます。

```bash
sbt run
```

実行すると、コンソールに挨拶が表示されます。

```text
Hello, World!
Scala 3 へようこそ！
```

## 📂 プロジェクト構成

```

│           └── Hello.scala  # メインプログラム
```
